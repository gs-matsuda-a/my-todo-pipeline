# base_template（Todo アプリ雛形）

**起動はするが未完成**の Todo アプリです（Node.js + Express + better-sqlite3）。
各章ではこれをコピーして作業フォルダを作り、Claude Code で完成させていきます。

```bash
npm install
npm start   # http://localhost:3000
```

- `server.js` … サーバ起動と `tasks` テーブル作成のみ（CRUD は未実装）
- `public/index.html` … UI の枠だけ（機能は未実装）

> このフォルダ自体は雛形なので編集しないでください。作業は各章でコピーした `my-todo-*` 内で行います。
